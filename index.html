⬇ 小窗启动器
把计时器做成一个干净小窗，浮在 PPT 放映之上（角落显示、自动计时、可半透明）。推荐方式，免插件、不收费。
一、装 PowerToys（只需一次，微软官方免费）
  1. 打开「Microsoft Store（微软商店）」→ 搜 PowerToys（发布者为 Microsoft）→ 安装。装完打开一次，让它在后台运行。
  2. 打开 PowerToys → 左侧「Always On Top（始终置顶）」→ 确认开关已开启；记下「激活快捷键」（默认 Win+Ctrl+T，有的机器是别的组合，以这里显示的为准，可自行改）。
  3. 建议在「常规」里打开「开机时启动」，以后无需手动开。
二、用小窗计时器
  1. 在本工具调好样式后点「小窗启动器」，下载一个 .bat 文件（会按当前配置自动命名，如「5分钟·橙·条状·小窗.bat」，链接已填好，无需编辑）。
  2. 双击该 .bat，会用 Edge / Chrome 的「应用模式」弹出一个干净小窗（无地址栏、无书签栏），自动开始计时。首次双击若有安全提示，点「仍要运行」。
  3. 拖小窗顶部标题条移动位置、拖边角缩放大小，摆到屏幕角落。
  4. 点中这个小窗 → 按 PowerToys 置顶快捷键（如 Win+Ctrl+T）→ 出现彩色边框即钉住，它会一直浮在最前面，包括盖在 PPT 之上。
  5. 之后正常放映 PPT，小窗始终浮在角落。再按一次快捷键可取消置顶。
放映须联网（小窗加载在线页面）。若 PPT「全屏放映」偶尔盖住小窗：在 Always On Top 设置里关掉「Game Mode 时不激活」，或把放映改为「幻灯片放映 → 设置 → 由观众浏览(窗口)」模式。置顶后还可按 Win+Ctrl+加/减号 调小窗透明度，更不挡内容。仅 Windows；Mac 请用下方「下载计时器」全屏播放。 提示：链接已把当前配置烤进 .bat，同一个进度条永远双击同一个 .bat 即可，不用每次重下；想要不同的进度条（换形态 / 时长 / 颜色），调好后再下一个即可。
⬇ 计时器
下载一个独立计时器网页，全屏单独播放（跨系统通用、无需联网、无需任何插件）。 ① 点「计时器」下载 HTML。② 双击用浏览器打开 → 按 F11（Mac 用浏览器全屏）进入全屏 → 点「开始计时」。③ 演讲到该环节时，从 PPT 切出来全屏播放，结束会闪烁提醒并响提示音；之后用 Alt+Tab（Mac 用 ⌘+Tab）切回 PPT。也可放副屏供自己控场。
🔗 复制嵌入链接
仅复制这个计时器的网址（已含全部配置，翻页即自动计时）。给进阶用户用：你可以把它粘到浏览器直接打开预览，或粘到支持嵌入网页的工具里。 日常用不到这个——把计时器放进 PPT，推荐直接用上方「小窗启动器」（下载即用、浮在 PPT 上）；或用「计时器」全屏播放。链接见下方框，点一下可全选复制。
